# RFC 4791 — CalDAV: Calendaring Extensions to WebDAV

- `deferred` — intentionally out of scope with reason

> **Context:** RFC 4791 is the CalDAV core specification — the `calendar-access` equivalent of RFC 6352 (CardDAV). It defines calendar collections, calendar object resources (iCalendar files), MKCALENDAR, the calendar-query and calendar-multiget reports, and the free-busy-query report. No test suite exists yet; this document serves as the implementation roadmap.
