## Coverage Summary

| Status | Count |
|--------|-------|
| conformance ✓ | 34 |
| lint only | 11 |
| not covered (testable) | ~6 |
| deferred | ~10 |

The 50 tests now cover: all named preconditions for PUT/COPY/MOVE, protected property enforcement, collation testing, unauthenticated access across four methods, non-existent property reporting, and group prefix matching in both prop selection and prop-filter. Remaining gaps: result truncation structure (R-91–R-93, not triggerable from client side), COPY/MOVE precondition tests for supported-address-data/valid-address-data (complex setup), and R-63/R-64/R-77.
