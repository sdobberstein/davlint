> **Context:** RFC 7809 allows CalDAV servers to omit `VTIMEZONE` components from iCalendar data when the time zone is a well-known standard zone available from an associated time zone distribution service (RFC 7808). Servers advertise this capability via `calendar-no-timezone` in the DAV header, and clients can request omission via the `CalDAV-Timezones: F` request header. This is an optional extension ‚Äî tests in this suite MUST be skipped if the server does not advertise `calendar-no-timezone` in its OPTIONS response.
