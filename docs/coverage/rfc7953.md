> **Context:** RFC 7953 defines the `VAVAILABILITY` iCalendar component for expressing when a calendar user is available. It extends RFC 5545 (iCalendar) and RFC 4791 (CalDAV). A `VAVAILABILITY` object can be stored as a calendar object resource in a CalDAV collection, and optionally as the value of the `CALDAV:calendar-availability` WebDAV property on a scheduling inbox. Several server requirements are **conditional on the server advertising `calendar-auto-schedule` support**.
