# RFC 9073 — Event Publishing Extensions to iCalendar

**Suite:** not yet created — blocked on rfc4791

Adds `PARTICIPANT`, `VLOCATION`, and `VRESOURCE` components to iCalendar for richer event publishing (venue details, resource booking, participant roles beyond organizer/attendee). Primarily relevant for public event publishing rather than personal calendar sync.

## Standalone suite verdict

**Low priority.** These components are not widely implemented in personal/team CalDAV servers (the primary target of davlint). Relevant mostly for event-publishing platforms. Blocked on rfc4791. Revisit after core CalDAV coverage is established and if a target server is known to implement these extensions.
