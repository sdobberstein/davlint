# RFC 5545 — iCalendar Format

**Suite:** not yet created

Defines the iCalendar data format. The server-testable surface is whether the server accepts valid iCalendar and rejects invalid iCalendar on PUT (returning `CALDAV:valid-calendar-data` or `CALDAV:valid-calendar-object-resource`). Tests depend on having an RFC 4791 calendar collection to PUT into.
