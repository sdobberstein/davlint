# RFC 7265 — jCal: The JSON Format for iCalendar

**Suite:** not yet created — blocked on rfc4791

Defines a JSON representation of iCalendar using the media type `application/calendar+json`. CalDAV servers MAY support this via content negotiation alongside `text/calendar`.

## Standalone suite verdict

**Low priority — same reasoning as rfc7095 (jCard).** jCal is a MAY; few production CalDAV servers implement it. Also blocked on rfc4791 for the CalDAV infrastructure. Skip unless a target server advertising jCal support is identified.
