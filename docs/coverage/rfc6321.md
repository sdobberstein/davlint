# RFC 6321 — xCal: An XML Representation for iCalendar

**Suite:** not yet created — blocked on rfc4791

Defines an XML representation of iCalendar using the media type `application/calendar+xml`. CalDAV servers MAY support this via content negotiation alongside `text/calendar`.

## Standalone suite verdict

**Low priority — same reasoning as rfc6351 (xCard) and rfc7265 (jCal).** xCal is a MAY; implementation is rare in production CalDAV servers. Blocked on rfc4791. Skip unless a concrete target server is identified.
